## Deploy no GitHub Pages

Este projeto está configurado para ser automaticamente implantado no GitHub Pages.

### Configuração Inicial

Para habilitar o GitHub Pages pela primeira vez, siga estes passos:

1. Acesse as configurações do repositório: `Settings` > `Pages`
2. Em "Build and deployment":
   - **Source**: Selecione "GitHub Actions"
3. Faça merge desta PR para a branch `main`
4. O workflow será executado automaticamente e o site será publicado

### Acesso

Após a configuração, o site estará disponível em:
`https://lfreire80.github.io/bblayout/`

### Atualizações Automáticas

Qualquer push para a branch `main` irá automaticamente atualizar o site no GitHub Pages.
